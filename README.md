# Ultimate Creators Club GitHub Pages Site

Upload these files to a public GitHub repository and enable GitHub Pages from the `main` branch root.

Files:
- `index.html` - public app website
- `terms.html` - Terms of Service
- `privacy.html` - Privacy Policy
- `style.css` - styling
- `.nojekyll` - keeps GitHub Pages from applying Jekyll processing

After publishing, your URLs will look like:

```text
https://YOUR-GITHUB-USERNAME.github.io/ultimate-creators-club/
https://YOUR-GITHUB-USERNAME.github.io/ultimate-creators-club/terms.html
https://YOUR-GITHUB-USERNAME.github.io/ultimate-creators-club/privacy.html
```

When TikTok gives you a URL prefix verification/signature file, upload that exact file into the same repository root as `index.html`.
